Fieldsets group related controls into logical sections.

## Basic usage

```twig
{{
    form.fieldset_start()
}}

    {# form fields #}

{{
    form.fieldset_end()
}}
```

## Helper options

You can configure this helper using the common helper options, the following options are specific to this helper.

| Option | Type | Description |
| ------ | ---- | ----------- |
| legend | string | Label for the optional `<legend>` element |

## Legend

Give fieldsets a descriptive legend to break your form into logical sections.


<div className="d-example">
  <fieldset>
    <legend className="legend">I am legend</legend>
    <div className="form__group">
      <label for="field-1" className="control__label">Example field</label>
      <div className="controls">
        <input id="field-1" name="field-1" type="text" className="form__control" />
      </div>
    </div>
  </fieldset>
</div>
