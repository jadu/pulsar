Creates a simple HTML link, often used inside other helpers.

<div className="d-example">
  <a href="http://www.jadu.net">Jadu</a>
</div>

## Basic usage

<Tabs
  groupID="example"
  defaultValue="twig"
  values={[
    { label: 'Twig', value: 'twig', },
    { label: 'HTML', value: 'html', },
  ]
}>
<TabItem value="twig">

```twig
{{
    html.link({
        'href': 'http://www.jadu.net',
        'label': 'Jadu'
    })
}}
```

</TabItem>
<TabItem value="html">

```html
<a href="http://www.jadu.net">Jadu</a>
```

</TabItem>
</Tabs>

## Helper options

You can configure this helper using the common helper options, the following options are specific to this helper.

| Option | Type | Description |
| ------ | ---- | ----------- |
| href | string | The URL location to navigate to |
| icon | string | An icon to place before the label |

## Accessibility

Refer to the [Jadu Accessibility Guidelines (internal)](http://accessibility-guide.ntn.jadu.net/components/links/) for detailed guidance on the accessibility requirements of this component.
