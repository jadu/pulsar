A basic helper to render a list of breadcrumb links. A view should set a `breadcrumbs` object which will then be rendered by the base template.

<div className="d-example">
  <nav className="breadcrumb" aria-label="breadcrumbs">
    <a href="#foo">foo</a> / <a href="#bar">bar</a> / <a href="#baz" aria-current="page">baz</a>
  </nav>
</div>

## Basic usage

<Tabs
  groupID="example"
  defaultValue="twig"
  values={[
    { label: 'Twig', value: 'twig', },
    { label: 'HTML', value: 'html', },
  ]
}>
<TabItem value="twig">

```twig
{%
    set breadcrumbs = [
        {'label': 'foo', 'href': '#foo'},
        {'label': 'bar', 'href': '#bar'},
        {'label': 'baz', 'href': '#baz'}
    ]
%}

{{ nav.breadcrumbs(breadcrumbs|default) }}
```

</TabItem>
<TabItem value="html">

```html
<nav class="breadcrumb" aria-label="breadcrumbs">
  <a href="#foo">foo</a> /
  <a href="#bar">bar</a> /
  <a href="#baz" aria-current="page">baz</a>
</nav>
```

</TabItem>
</Tabs>
