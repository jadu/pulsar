About Pulsar

Pulsar provides a common framework, tools and documentation to help designers and developers build user interfaces within the Jadu platform.

Designed for Jadu

Components are designed with accessibility, usability and responsiveness in mind, providing additional features requiring as little configuration as possible.

Collaborate

We welcome any feedback and ideas on how to improve the design system. If you are interested in contributing you can check out our contribution guidelines or report an issue.
